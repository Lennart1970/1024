# Brain Sources

Shared training and reference material for bots working from this workspace.

## Purpose

This directory contains durable source material that agents can use for:
- training design
- prompt design
- workflow design
- evaluation criteria
- product reasoning

These files are **knowledge sources**, not automatic model fine-tuning artifacts.

## Discovery rule for bots

When looking for reusable training/reference material, check:
1. `TRAINING_SOURCES.md`
2. `brain-sources/README.md`
3. files inside `brain-sources/`

## Available sources

### `decision-quality-agentic-model.md`
- Imported from a user-provided `.docx`
- Theme: move from information output to better decisions via qualified signals
- Key concepts:
  - Brain
  - Lens
  - Perception
  - Interpretation
  - Decision & Execution
  - HITL / Learning

## Usage guideline

When using a source from this directory, agents should:
- cite the file name when useful
- distinguish source material from their own synthesis
- avoid claiming the file was used for literal model training unless that actually happened
